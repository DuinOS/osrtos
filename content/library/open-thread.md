OpenThread released by Nest is an open-source implementation of the Thread networking protocol.

<!--more-->

### Features
- IPv6 configuration and raw data interface
- UDP sockets
- CoAP client and server
- DHCPv6 client and server
- DNSv6 client
- NCP support
- Border router
